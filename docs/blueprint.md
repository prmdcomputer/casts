# **App Name**: Citizen Certificate

## Core Features:

- HTML Form Rendering: Generates and renders a certificate application form based on the provided HTML code.
- Input Field Extraction: Extracts relevant input fields from the HTML form code for certificate application. Determines input type (text, dropdown, radio) and validation requirements.
- Data Submission and Handling: Allows users to input data into the form fields, handle form submission, and perform client-side validation using extracted validation parameters.
- Data Transformation: The tool attempts to understand which inputs represent personal information and which represent metadata, and formats/cleanses personal information like name, address, and family details in a consistent, globally recognizable manner.
- Certificate Preview: Generates a preview of the certificate based on the submitted data, mirroring the layout of the final certificate (as shown in 2nd HTML source) but substituting AI-generated placeholder data
- PDF generation: Converts the certificate preview to PDF format for printing or sharing.

## Style Guidelines:

- Primary color: Deep blue (#30475E), evoking trust, authority, and official documentation.
- Background color: Light gray (#F2F4F7), providing a clean and professional backdrop.
- Accent color: Teal (#3AAFA9), used sparingly for highlights and interactive elements to draw attention without overwhelming the design.
- Body font: 'Inter', a sans-serif font providing excellent readability for form labels and informational text.
- Headline font: 'Space Grotesk', sans-serif, for a clean, modern style for titles, headers, and calls to action.
- Use a consistent set of government-themed icons, such as a document or seal to reinforce credibility.
- Employs a clean, organized layout with clear visual hierarchy to guide users through the form-filling process, ensuring accessibility and ease of use.
